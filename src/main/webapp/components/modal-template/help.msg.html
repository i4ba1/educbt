  10. EMAIL : isilah email dengan format teks
  11. MASIH_AKTIF : isilah dengan "ya" atau "tidak"

Format Pengisian Impor Data Bank Soal

  1. Tipe Soal wacana ? : Ceklis jika inggin membuat soal "wacana", dan biarkan tidak terceklis jika ingin membuat soal "pilihan ganda" atau "benar-salah"
  2. Teks Wacana : isilah wacana yang nantinya akan dijadikan wacana pada soal yang diimpor.
  3. XLS Template : tombol untuk mengunduh file template bank soal
  4. Cara Pengisian Template
    • NAMA SOAL : Nama soal bisa diisi dengan huruf maupun angka , contoh : Soal 1
    • PERTANYAAN : isilah dengan format teks, dengan langkah :
      • Klik kanan "Cell" yang aktif
      • Pilih "Format Cells"
      • pada "Category" pilih "Text"
      • OK
    • PILIHAN A-E : isilah dengan format teks
    • KUNCI JAWABAN : isilah kunci jawab dengan format ( A - E) untuk tipe "pilihan ganda", atau (1 = benar, 0 = salah) untuk tipe pertanyaan "benar-salah"
    • KESULITAN : tingkat kesulitan dibagi jadi tingkatan yaitu 1(mudah),2(sedang), atau 3(sukar)
    • PEMBAHASAN : isilah pembahasan dengan format teks
    • JENIS_SOAL : isilah jenis soal dengan kode "MC" untuk soal tipe pilihan ganda, atau kode "TF" untu soal tipe "benar-salah"
    • Disarankan menghapus 3-5 baris kosong setelah soal terakhir

  5. Pilih Berkas : pilihlah berkas bank soal, sesuai format template yang disediakan
  6. Pilih Simpan
